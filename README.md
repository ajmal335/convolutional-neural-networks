# 🧠 Convolutional Neural Networks (CNN)

This repository contains implementations of Convolutional Neural Networks for image classification tasks.

## 🔍 Overview
The project demonstrates how CNNs can be used to extract features from images and perform classification tasks.

## 📌 Implementations

### 1. CNN on CIFAR-10 Dataset
- Built a CNN model to classify images into 10 categories
- Used convolution, pooling, and dense layers
- Applied normalization and model evaluation

## ⚙️ Key Concepts
- Convolution layers
- Pooling layers
- Activation functions (ReLU, Softmax)
- Image preprocessing

## 🛠️ Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib

## 📈 Results
The CNN model achieved strong accuracy on image classification tasks and outperformed basic ANN models.
